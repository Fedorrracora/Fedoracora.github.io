Задача № 1

На вход получается одно целое число: -1; 0; 1.

Нужно сделать так, чтобы если на вход получается число -1 или 1, то программа должна выдать число 0, а если на вход идёт число 0, то команда должна выдать 1.


Sample Input:
0


Sample Output:
1


Sample Input:
1


Sample Output:
0


Sample Input:
-1


Sample Output:
0
